2.Vector Graphics:- CorelDraw

Coreldraw is a vector graphics editor developed and marketed by alludo. it is also the name of the Corel graphics suite, which includes the bitmap-image editor Corel photo-paint as well as other graphics related program.
i have used Coreldraw Graphics suite X7 for image processing. Using this software, I have transformed a regular image into one suitable for laser engraving.

This is the interface of CorelDraw software. click on file and create a new Document. I used image of my favourite character From marval.

By right clicking on workspace or by using Ctrl+i key import your jpeg image.

Then go to Bitmaps tab, select mode and select Grayscale(8-bit) after this you can see your original image become Black and white.

Again go to Bitmaps tab, select mode and select Black and White(1-bit) as shown in image and export the file with BMP extenstion.
